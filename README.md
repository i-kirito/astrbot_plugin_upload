# AstrBot 插件上传安装器

> 本项目是基于 AstrBot 开发的插件，用于通过消息命令安装/卸载插件

## 功能

- 上传 ZIP 文件安装插件
- 从本地目录安装插件
- 卸载已安装的插件

## 配置

| 配置项 | 说明 |
|--------|------|
| admin_only | 是否限制仅管理员使用（默认 true） |
| astrbot_url | AstrBot API 地址（默认 http://localhost:6185） |
| api_username | AstrBot API 用户名（默认 astrbot） |
| api_password_md5 | AstrBot API 密码的 MD5 值 |

## 指令

| 指令 | 说明 |
|------|------|
| `/上传插件` | 上传 ZIP 文件安装插件 |
| `/安装插件 <路径>` | 从本地目录安装插件 |
| `/卸载插件 <名称>` | 卸载已安装的插件 |
| `/插件帮助` | 显示帮助信息 |

## 使用说明

1. **上传安装**：发送 `/上传插件` 命令并附带 ZIP 文件
2. **本地安装**：`/安装插件 /path/to/plugin_dir`
3. **卸载插件**：`/卸载插件 plugin_name`

## 注意事项

- 仅管理员可以使用安装/卸载功能
- 插件 ZIP 文件需要包含完整的插件目录结构
- 需要正确配置 API 密码才能使用安装功能
